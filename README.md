## 💫 About Me

I am a multi-disciplinary software engineer specializing in performance applications, modern full-stack web architectures, and specialized hardware/software integrations. My engineering philosophy focuses on writing clean, annotated code, optimizing speed using low-level programming, and integrating complex toolchains in an accessable format. I build software from the ground up that minimizes overhead and maximizes computational throughput while giving as much power to the user as possible.

- ⚙️ **Low-Level & Game Systems:** Deep experience in native system engineering using **C**, **C++**, and **C#**, leveraging automation scripting via **Batch**, standalone compilation toolchains (**Clang**, **Code::Blocks**, **Visual Studio**), and cross-platform compilation with **Unity**.
- 🌐 **Full-Stack & Interactive Web:** Proficient in lightweight, structured web applications using **JavaScript**, **HTML5**, and **CSS3**, backed by responsive, data-driven vector graphics.
- 🗄️ **Data Architecture & Technical Docs:** Fluent in structured data workflows utilizing **XML**, relational database design (**PostgreSQL**, **SQL**) as well as programmatic design documentation workflows utilizing **Markdown**, **LaTeX**, and **Mermaid** graphing.
- 🎨 **Creative Design & Physical Prototyping:** Passionate about advanced asset pipelines using **Blender** for 3D modeling, **FreeCAD** for mechanical engineering, **Bambu Lab** additive manufacturing workflows, **GIMP** for texture workflows, **MuseScore** for music and sound composition, and font engineering via **FontForge**.
